# Livexshoppro

แพลตฟอร์ม Live Shopping สำหรับการขายสินค้าแบบ Real-time

## เกี่ยวกับโปรเจค

Livexshoppro เป็นแพลตฟอร์มสำหรับการไลฟ์ขายสินค้าออนไลน์ ที่รวมฟีเจอร์การถ่ายทอดสด การจัดการสินค้า และระบบการสั่งซื้อไว้ในที่เดียว

## โครงสร้างโปรเจค

```
Livexshoppro/
├── src/              # Source code หลัก
│   ├── components/   # UI Components
│   ├── pages/        # หน้าต่างๆ
│   ├── services/     # Business logic และ API
│   ├── utils/        # Utility functions
│   └── types/        # TypeScript types
├── public/           # Static assets
├── tests/            # Test files
└── docs/             # Documentation
```

## การติดตั้ง

```bash
# Clone repository
git clone <repository-url>
cd Livexshoppro

# ติดตั้ง dependencies
npm install

# รันในโหมด development
npm run dev
```
